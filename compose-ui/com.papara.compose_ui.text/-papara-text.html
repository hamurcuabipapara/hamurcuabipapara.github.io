PaparaText

@Composable
fun PaparaText(modifier: Modifier = Modifier, text: String? = null, annotatedString: AnnotatedString? = null, color: Color = Color.Companion.Unspecified, fontSize: TextUnit = TextUnit.Companion.Unspecified, fontStyle: FontStyle? = null, fontWeight: FontWeight? = FontWeight.Companion.Normal, fontFamily: FontFamily = PaparaFontFamily, letterSpacing: TextUnit = TextUnit.Companion.Unspecified, textDecoration: TextDecoration? = null, textAlign: TextAlign? = null, lineHeight: TextUnit = TextUnit.Companion.Unspecified, overflow: TextOverflow = TextOverflow.Companion.Clip, softWrap: Boolean = true, singleLine: Boolean = false, maxLines: Int = if (singleLine) 1 else Int.Companion.MAX_VALUE, minLines: Int = 1, onTextLayout: (TextLayoutResult) -> Unit = {}, style: TextStyle = LocalTextStyle.current)

A customizable text composable that wraps the built-in Text composable.

Parameters

modifier

The modifier to be applied to the text.

text

The text to be displayed. If null, annotatedString will be used instead.

annotatedString

The annotated string to be displayed. If null, an empty annotated string will be used.

color

The color of the text. Defaults to Color.Unspecified.

fontSize

The font size of the text. Defaults to TextUnit.Unspecified.

fontStyle

The font style of the text (e.g. italic, bold, etc.). Defaults to null.

fontWeight

The font weight of the text (e.g. normal, bold, etc.). Defaults to FontWeight.Normal.

fontFamily

The font family of the text. Defaults to PaparaFontFamily.

letterSpacing

The letter spacing of the text. Defaults to TextUnit.Unspecified.

textDecoration

The text decoration of the text (e.g. underline, strikethrough, etc.). Defaults to null.

textAlign

The text alignment of the text (e.g. left, center, right, etc.). Defaults to null.

lineHeight

The line height of the text. Defaults to TextUnit.Unspecified.

overflow

The text overflow behavior (e.g. clip, ellipsis, etc.). Defaults to TextOverflow.Clip.

softWrap

Whether the text should wrap to the next line when it reaches the end of the line. Defaults to true.

singleLine

Whether the text should be displayed on a single line. Defaults to false.

maxLines

The maximum number of lines the text should occupy. Defaults to Int.MAX_VALUE if singleLine is false, and 1 if singleLine is true.

minLines

The minimum number of lines the text should occupy. Defaults to 1.

onTextLayout

A callback function that will be called when the text layout is calculated.

style

The text style to use. Defaults to the current LocalTextStyle.
